TenderPaws Pet Adoption

EJS | JavaScript | Node | Express | MongoDB | Passport

A fully responsive website that simplifies the pet adoption process and connects pet lovers with furry companions. Originally created as a landing page, it has now been converted from HTML to EJS and extended into a full-stack website using Node.js, Express.js, and MongoDB. With the use of Passport.js, it includes user registration and secure authentication, allowing users to perform operations for creating and managing an extensive list of pet listings.
